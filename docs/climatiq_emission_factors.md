# Climatiq Emission Factors for Recycling

This document contains verified API snippets for calculating carbon emissions saved/generated by recycling various materials.

## 1. Tires

**ID**: `waste-type_tires-disposal_method_recycled`  
**Region**: US  
**Source**: EPA

```bash
curl --request POST \
    --url https://api.climatiq.io/data/v1/estimate \
    --header "Authorization: Bearer $CLIMATIQ_API_KEY" \
    --data '{
	"emission_factor": {
		"activity_id": "waste-type_tires-disposal_method_recycled",
		"source": "EPA",
		"region": "US",
		"year": 2025,
		"source_lca_activity": "end_of_life",
		"data_version": "^29"
	},
	"parameters": {
		"weight": 80,
		"weight_unit": "t"
	}
}'
```

## 5. Plastics

**ID**: `plastics_rubber-type_plastics_recycled`  
**Region**: DE (Germany)  
**Source**: Bafa  
_Note: This factor uses `cradle_to_gate` LCA activity and requires `allowed_data_quality_flags`._

```bash
curl --request POST \
    --url https://api.climatiq.io/data/v1/estimate \
    --header "Authorization: Bearer $CLIMATIQ_API_KEY" \
    --data '{
	"emission_factor": {
		"activity_id": "plastics_rubber-type_plastics_recycled",
		"source": "Bafa",
		"region": "DE",
		"year": 2025,
		"source_lca_activity": "cradle_to_gate",
		"data_version": "^29",
		"allowed_data_quality_flags": [
			"notable_methodological_variance"
		]
	},
	"parameters": {
		"weight": 80,
		"weight_unit": "t"
	}
}'
```
